fiber
This code computes the atomic fibers and the extended atomic fibers of a matrix. See
  • R. Hemmecke. Computation of Atomic Fibers of Z-Linear Maps. e-print arXiv:math.CO/0410289, 2004.
for more details on the algorithm implemented.

Usage 

./fiber [--options] fileName

Example 

./fiber --quiet fileName

Option Effect
--quiet no output is written to the screen

Output file Content
fileName.efib list of extended atomic fibers and minimal elements in fibers
fileName.fib list of atomic fibers and minimal elements in fibers
