'graver' computes (truncated) Graver bases of lattices L where L is given
as ker(A) for given matrix A
or
by lattice generators.
Graver basis from matrix

If present, lattice generators will be read automatically from "fileName.lat".

If present, upper bound vector will NOT be read automatically from "fileName.tru".

./graver fileName
Truncated Graver basis from matrix

If present, lattice generators will be read automatically from "fileName.lat".

Needs upper bounds in "fileName.tru".

./graver tru fileName
Symmetric Graver basis from matrix

Needs file "fileName.sym"!

./graver sym fileName
Note that the option 'inf 0' surpresses most of the output of 'graver'.
