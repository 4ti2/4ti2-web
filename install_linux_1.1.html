4ti2 installation guide for linux machines
  • Create a directory for 4ti2, call it '4ti2', say.

  • Download the file for linux machines to the new directory '4ti2'.

  • Go to directory '4ti2'.

  • Unzip file by 'gunzip 4ti2_linux_1.1.tar.gz'.

  • Untar file by 'tar xvf 4ti2_linux_1.1.tar'.

  • Change attributes to 700 by 'chmod 700 *'.

  • Remove tar file by 'rm 4ti2_linux_1.1.tar'.

  • See tutorial on how to use 4ti2 for your computations.
