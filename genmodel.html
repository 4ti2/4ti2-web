genmodel
This code computes the problem matrix corresponding to a graphical statistical model given by a simplicial complex and levels on the nodes in a file, say 'foo.mod'.

Usage 

./genmodel [--options] foo

Example

Consider the problem of 3x3x3 tables with 2-marginals. These are given by K_3 as the simplicial complex on 3 nodes and with levels of 3 on each node.

In '333.mod' write 

3
3 3 3
3
2 1 2
2 2 3
2 3 1

./genmodel 333
