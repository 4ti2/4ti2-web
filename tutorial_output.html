Hilbert bases Graver bases Gröbner bases Markov bases Minimization
'hilbert', 'graver', 'groebner', and 'markov' produce output files that contain a list of vectors in input format.

To change that list to a format that suits you better,
or to extract certain subsets of vectors,
or to extract more information about vectors,
use the function 'output'.
List of vectors in Maple format ./output map fileName
List of binomials in variables x[1],...,x[n] ./output bin fileName
List of extreme rays (from non-truncated Hilbert bases) ./output ray fileName
List of 0-1 vectors ./output 0-1 fileName
List of vectors as 3-way tables of sizes (x,y,z), (Hint: Use z = 1 for 2-way tables) ./output 3way x y z fileName
Transpose of matrix (list of vectors) ./output tra fileName
L1-norms of vectors (Hint: Vectors with max norm will be printed to screen!) ./output deg fileName
Extracts lex-max representative under given symmetry (needs file fileName.sym!) ./output sym fileName
List of positive parts ./output pos fileName
